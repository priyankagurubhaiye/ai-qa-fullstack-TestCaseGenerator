# 🤖 AI-Powered QA Test Case Generator

## 🚀 Overview
This project demonstrates an AI-first QA approach using GenAI to:
- Automatically generate test cases from requirements
- Identify edge cases and negative scenarios
- Improve test coverage and reduce manual effort

## 🧠 Features
- AI-based test case generation
- Prompt engineering for QA workflows
- Scalable automation-ready structure
- Supports positive, negative, and edge case scenarios

## 🛠️ Tech Stack
- Python
- OpenAI API
- Pytest

## ▶️ How to Run

1. Install dependencies:
pip install -r requirements.txt

2. Add your OpenAI API key in:
config/config.py

3. Run:
python main.py

## 🎯 Use Case
- AI-first QA automation
- Requirement-to-test-case conversion
- Faster release cycles with better coverage
